🔒 Política de Privacidade

Mestres do Cálculo  |  Última atualização: 19 de dezembro de 2025

100% Offline Zero Coleta de Dados Sem Anúncios Sem Rastreamento COPPA ✓ LGPD ✓ GDPR ✓

🔒 Resumo (TL;DR)

O Mestres do Cálculo NÃO coleta NENHUM dado pessoal.

  • Funciona 100% offline
  • Zero coleta de dados
  • Sem anúncios de terceiros
  • Sem rastreamento
  • Sem compras dentro do app

1. Introdução

Bem-vindo ao Mestres do Cálculo, um aplicativo educativo de matemática desenvolvido especialmente para crianças de 6 a 12 anos.

Esta Política de Privacidade descreve como tratamos informações no aplicativo. Levamos a privacidade das crianças muito a sério e estamos comprometidos em proteger os dados de todos os nossos usuários.

2. Informações que NÃO Coletamos

Nosso aplicativo foi projetado para funcionar totalmente offline e não coleta, armazena ou transmite nenhum dos seguintes dados:

  • Nomes ou informações de identificação pessoal
  • Endereços de e-mail
  • Números de telefone
  • Localização geográfica (GPS)
  • Fotos ou vídeos
  • Dados de contatos
  • Identificadores de dispositivo
  • Endereços IP
  • Histórico de navegação
  • Informações de login ou senhas

Garantimos que NENHUMA informação pessoal ou identificável é coletada, processada ou compartilhada.

3. Armazenamento Local de Dados

O que é armazenado localmente

O aplicativo salva apenas as seguintes informações no dispositivo:

Progresso do Jogo

  • Tabuadas completadas
  • Pontuações alcançadas
  • Estrelas conquistadas
  • Troféus desbloqueados

Preferências de Configuração

  • Volume da música de fundo
  • Volume dos efeitos sonoros
  • Ativação/desativação de vibração

Onde esses dados ficam

Todos os dados são armazenados exclusivamente no dispositivo usando a tecnologia Hive (banco de dados local). Estes dados:

  • NÃO são enviados para servidores externos
  • NÃO são compartilhados com terceiros
  • NÃO são usados para análise ou publicidade
  • Permanecem completamente privados no dispositivo
  • São deletados quando o aplicativo é desinstalado

4. Permissões do Aplicativo

Permissões solicitadas

Vibração (VIBRATE)

Finalidade: Fornecer feedback tátil quando a criança toca nos botões ou acerta/erra uma resposta.
Acesso a Dados: Esta permissão NÃO permite acesso a nenhum dado pessoal.

Internet (INTERNET) – Opcional

Finalidade: Carregar fontes do Google Fonts para melhorar a aparência do texto.
Observação: OPCIONAL. Se o dispositivo estiver offline, o aplicativo funciona normalmente com fontes padrão.
Acesso a Dados: NÃO usamos esta permissão para enviar ou receber dados do usuário.

Permissões que NÃO solicitamos

  • Câmera
  • Microfone
  • Localização (GPS)
  • Contatos
  • Armazenamento de mídia
  • Calendário

5. Conformidade com COPPA, LGPD e GDPR

COPPA (Children's Online Privacy Protection Act)

  • Não coletamos informações pessoais de crianças menores de 13 anos
  • Não compartilhamos nenhuma informação com terceiros
  • Não exibimos anúncios de redes de publicidade
  • Não usamos recursos de rastreamento ou análise comportamental
  • Implementamos Parental Gate em áreas sensíveis

LGPD (Lei Geral de Proteção de Dados – Brasil)

  • Base legal: Não há tratamento de dados pessoais
  • Transparência: Esta política descreve claramente nosso não-tratamento de dados
  • Segurança: Não há dados para serem comprometidos
  • Direitos do titular: Não aplicável, pois não há dados pessoais

GDPR (Regulamento Geral – União Europeia)

  • Não processamos dados pessoais
  • Não usamos cookies ou rastreadores
  • Não transferimos dados para fora da UE (pois não há dados)

6. Publicidade e Terceiros

Sem anúncios

O Mestres do Cálculo é 100% livre de anúncios. Não exibimos:

  • Anúncios de redes de publicidade (Google Ads, Facebook Ads, etc.)
  • Banners
  • Vídeos promocionais
  • Links patrocinados

Sem SDKs de rastreamento

  • Google Analytics
  • Firebase Analytics (exceto Crashlytics anônimo – veja seção 7)
  • Facebook SDK
  • Mixpanel, Amplitude, etc.

Sem compras no app

  • Compras dentro do app (In-App Purchases)
  • Assinaturas
  • Upgrades pagos
  • Loot boxes ou mecânicas de pagamento

7. Relatórios de Erros (Crash Reports)

Somente se o aplicativo travar, podemos coletar automaticamente informações técnicas e anônimas:

  • Modelo do dispositivo (ex: Samsung Galaxy S21)
  • Versão do Android (ex: Android 12)
  • Versão do aplicativo (ex: 1.0.0)
  • Stack trace do erro

Usamos Firebase Crashlytics configurado para coletar apenas dados técnicos anônimos.

Você pode optar por não enviar relatórios de erro: Configurações > Google > Uso e Diagnóstico > Desativar.

8. Segurança

  • Armazenamento criptografado de progresso local (Hive)
  • Código ofuscado para prevenir manipulação
  • Sem backdoors ou acesso remoto
  • Parental Gate em áreas sensíveis

9. Direitos dos Pais/Responsáveis

Como não coletamos dados pessoais, não há informações para acessar, corrigir, excluir ou exportar. No entanto, os pais/responsáveis podem:

  1. Revisar o progresso local: Acessar a tela de troféus e estatísticas
  2. Apagar o progresso: Usar o botão "Limpar Todo Progresso" nas configurações
  3. Desinstalar o app: Remove todos os dados locais automaticamente

10. Alterações a Esta Política

Podemos atualizar esta Política de Privacidade ocasionalmente. Notificaremos sobre alterações significativas:

  • Na página de descrição do aplicativo na Play Store
  • Dentro do próprio aplicativo (em caso de atualizações que afetem privacidade)

A data da "Última atualização" no topo deste documento indica quando a política foi revisada pela última vez.

11. Declaração Final

Desenvolvemos o Mestres do Cálculo com o mais alto respeito pela privacidade das crianças. Nossa filosofia é simples:

"O melhor dado é o que não coletamos."

Acreditamos que aplicativos educativos para crianças devem:

  • Focar no aprendizado, não em dados
  • Respeitar a privacidade desde o design (Privacy by Design)
  • Ser transparentes com pais e educadores
  • Funcionar offline sempre que possível
