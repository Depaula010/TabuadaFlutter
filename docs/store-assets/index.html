📸 Como exportar as imagens

  1. Abra o arquivo desejado no Google Chrome
  2. Abra o DevTools: F12 → aba Device Toolbar (Ctrl+Shift+M)
  3. No campo de dimensões, digite exatamente 1080 × 1920 (screenshots) ou 1024 × 500 (feature graphic)
  4. Zoom em 100% (Ctrl+Shift+P → "Set zoom" → 100%)
  5. Ctrl+Shift+P → digite Capture screenshot → Enter
  6. O PNG é salvo automaticamente em Downloads ✅
